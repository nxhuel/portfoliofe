Sobre Mí

Estudiante de Análisis de Sistemas con experiencia en desarrollo de software. Me especializo en arquitecturas backend robustas usando C#, ASP.NET Core, Entity Framework Core como ORM, SQL Server como motor de base de datos y Azure para su despliegue. Trabajo de manera eficiente y ágil, siguiendo metodologías ágiles como Scrum, principalmente bajo el paradigma de POO, respetando los principios SOLID y buenas prácticas de seguridad, con el objetivo de brindar sistemas escalables y mantenibles.

También tengo conocimientos en Angular, aprovechando su arquitectura modular, formularios reactivos, uso de pipes y componentes reutilizables, integrando interfaces limpias con Bootstrap.

Participo activamente en el relevamiento y documentación de sistemas, aplicando análisis funcional y diagramas UML.
